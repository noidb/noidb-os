# NOID-B OS V2

실제로 Vercel에 배포 가능한 Next.js 프로젝트입니다.

## 현재 기능
- 모바일/노트북 반응형
- 제품사진 업로드 및 미리보기
- 거래처 부산 기본값
- 써지컬스틸 기본값
- 로즈골드,골드,실버 기본값
- 모델명 자동 생성
- 쿠팡 상품명 자동 생성
- 검색태그 10개 자동 생성
- 상품정보 JSON 저장

## Vercel 설정
현재 GitHub 저장소에서 이 프로젝트를 `noidb-os-v2` 폴더로 올렸다면,
Vercel의 Root Directory를 `noidb-os-v2`로 지정하세요.
